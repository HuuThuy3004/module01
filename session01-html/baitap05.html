Đây là trang web của tôi

Chào mừng đế trang web của tôi. Đây là một ví dụ về cách sử dụng thẻ img để hiển thị hình ảnh trên trang web.

Đây là một hình ảnh lấy từ trên web Pinterest.com. Bạn có thể thay đổi hình ảnh bằng cách dùng thuộc tính height và width , phần mô tả bạn điền vào chỗ thuộc tính alt là được
